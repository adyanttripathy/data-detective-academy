# 🕵️ Data Detective Academy

A data-analytics learning game for all ages, built as a single-file React component.

Crack 8 randomly generated mini-cases that teach real analytics skills:

- **Chart reading** — compare bars and measure gaps
- **Averages** — mean, median, and when outliers make the mean lie
- **Outlier hunting** — tap the dot that doesn't belong on a scatter plot
- **Trend spotting** — see the direction through the noise
- **Correlation vs. causation** — bust classic traps like "ice cream causes sunburns"

Three difficulty ranks (Rookie 7+, Sleuth 11+, Chief for teens/adults), streak bonuses,
"Data Bite" explanations after every answer, and a score that's drawn live as a line chart.

## Running it

`data-detective.jsx` is a self-contained React component (default export, no required props).
Drop it into any React app, or paste it into an artifact-style React sandbox. Charts are
inline SVG — no chart libraries needed. Styling uses Tailwind utility classes plus inline styles.
